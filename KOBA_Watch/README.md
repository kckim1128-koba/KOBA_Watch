# 📈 KOBA Watch

> Keep Observing, Build Assets.

KOBA Watch는 미국 주식 시장을 자동으로 분석하여
매일 아침 투자 브리핑을 카카오톡으로 보내주는 프로젝트입니다.

---

# 현재 기능

✅ 미국 주식 시세 조회

✅ RSI 계산

✅ 목표가까지 거리 계산

✅ VIX / SOX 확인

✅ AI 의견 생성

✅ 카카오톡 전송

---

# 프로젝트 구조

KOBA_Watch

config.py
→ 설정

market.py
→ 시세 조회

indicator.py
→ RSI 계산

news.py
→ 뉴스

ai.py
→ AI 의견

send_kakao.py
→ 메시지 생성 및 카카오톡 발송

main.py
→ 프로그램 시작
